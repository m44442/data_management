## 【スライド1】タイトル（30秒）

皆さん、こんにちは。松本陸です。

e-Stat政府統計を用いた3時点時系列分析により、大型商業施設が地域小売業に与える影響を定量的に分析しています。

---

## 【スライド2】研究背景・目的（1分）

まず、研究背景についてご説明します。

2000年代以降、イオンモールなどの大型商業施設が地方都市に急速に展開しました。これにより既存の商店街や小売業への影響が社会問題となっています。

しかし、これまでの研究では統計的根拠に基づく定量的な影響分析が限定的でした。

そこで本研究では、**e-Stat政府統計を用いた時系列分析により、大型商業施設が地域小売業に与える影響を定量的に解明する**ことを目的としています。

研究の意義は3点あります。

- 第一に、e-Statの複数データソース（CSV・API）を組み合わせた高精度データ分析
- 第二に、3時点（2007年、2012年、2021年）の長期変化を追跡する時系列分析
- 第三に、全国5地域から10都市を選定した網羅的な比較分析

です。

---

## 【スライド3】分析手法（2分）

次に、分析手法についてご説明します。

### データソース

データソースは、**e-Stat、政府統計の総合窓口**です。
経済センサスの2007年、2012年、2021年のCSVデータ、および2014年のAPIデータを使用しました。

**データ収集の特徴**:

- CSVデータ（2007/2012/2021）：市区町村レベルの集計値
- APIデータ（2014）：超詳細地域コードレベルの詳細データ（参考値として取得）
- 主分析ではCSVの3時点データを使用

### 対象都市

Phase 1として**10都市**を選定しました。

選定理由は3つあります：

1. 全国5地域、関東、中部、近畿、四国、九州をバランスよくカバー
2. 大型商業施設の開業時期が明確
3. e-Statでのデータ取得確実性が高い

代表的な都市としては、

- 関東から越谷市（イオンレイクタウン、2008年開業、日本最大級）
- 中部から新潟市、岐阜市
- 九州から熊本市、大分市、佐賀市、鹿児島市
  などです。

### 分析指標

分析指標は4つです：

- 事業所数
- 従業者数
- 年間商品販売額
- 売場面積

### 統計分析手法

統計分析手法は**6種類**実施しています：

1. **記述統計分析** - 平均値、中央値、標準偏差の算出
2. **時系列分析** - 年平均成長率の算出とトレンド検定
3. **地域比較分析** - ANOVA（分散分析）による地域間差異検定
4. **クラスター分析** - K-means法による都市類型化
5. **相関分析** - 各指標間の相関係数算出
6. **効率性分析** - 事業所当たり従業者数などの効率性指標

### システム構成

システム構成は、TypeScriptでデータ収集と統合を行い、Pythonで統計分析と可視化を実行するという、2段階のパイプラインです。

## 【スライド4】収集データの概要（2分）

続いて、収集データの概要です。

### データ収集状況

現時点で、

- **総レコード数は28レコード（CSV年度のみ）**
- 対象都市数は10都市
- 対象年次は3時点（2007/2012/2021）
- データ完全性は**93%**（28/30レコード）

を達成しています。

**注記**: 2014年APIデータ（10レコード）も取得していますが、集計レベルが異なるため、主分析からは除外しています。

### データソースの特性

データソースには2種類あります：

- **CSVデータ（2007/2012/2021年）**: 市区町村レベルの集計データ
- **APIデータ（2014年）**: 超詳細地域コードレベルのデータ

**重要な注意点**: 2014年APIデータは超詳細地域コード単位で取得されているため、市区町村レベルのCSVデータと直接比較できません。

例えば、越谷市の場合：

- CSV年度：約1,600～2,000事業所（市区町村単位）
- API 2014年：14,500事業所（超詳細地域コード単位）

このため、**本分析では主にCSV年度（2007/2012/2021）の3時点比較**を行っています。

### 基本統計量（CSV年度ベース）

CSVデータ（2007/2012/2021年）の基本統計量は以下の通りです：

- 事業所数の平均は約3,200事業所、範囲は1,523から7,854事業所
- 従業者数の平均は約25,000人、範囲は12,608人から52,095人
- 販売額の平均は約4,800億円
- 売場面積の平均は約568,000平方メートル

**注**: これらはCSV年度のみの統計値です。2014年APIデータは集計レベルが異なるため除外しています。

### 地域別分布（CSV年度ベース）

地域別では、

- **中部地区が最も事業所数が多く、平均約4,700事業所**
- 九州、関東と続き
- **近畿地区が最も少なく、平均約1,900事業所**

となっており、約**2.5倍の地域格差**が確認されています。

## 【スライド5】分析結果（途中経過）（3分）

それでは、現時点での分析結果をご報告します。

### 1. 時系列分析結果

まず、時系列分析です。

**事業所数は2007年から2012年にかけて全都市で平均約30%減少**しています。

- 高松市では31.7%減（4,103から2,802事業所）
- 宮崎市では30.6%減（3,814から2,647事業所）
- 越谷市では18.4%減（1,984から1,618事業所）

このことから、**2007年から2012年に構造的変化が発生した**と考えられます。

一方、2012年から2021年にかけては、都市により増減が分かれました。

**従業者数については都市により明暗が分かれました。**

越谷市の例を見ると：

- 2007年：事業所数1,984、従業者数16,165人
- 2012年：事業所数1,618、従業者数14,176人
- 2021年：事業所数1,679、従業者数17,382人

**事業所数は15%減少したものの、従業者数は7.5%増加**しています。

このことから、**事業所の大型化・効率化が進行中**であることが示唆されます。

### 2. 地域比較分析結果

次に、地域比較分析です。

地域間格差を定量化するため、変動係数を算出しました（CSV年度ベース）。

- 事業所数の変動係数は**約50%**
- 従業者数の変動係数は**約42%**

いずれも**中～高程度の格差**です。

地域別特性としては、

- 中部地区が平均事業所数約4,700事業所
- 近畿地区が平均事業所数約1,900事業所
- 格差比は約2.5倍

となっており、**地域特性に応じた政策が必要**であることが分かりました。

### 3. 効率性分析結果

効率性分析では、事業所当たり従業者数の推移を分析しました。

- 2007年：平均約7.2人/事業所
- 2021年：平均約9.0人/事業所
- **増加率は約25%**

このことから、**小規模事業所の淘汰と中規模化が進行している**ことが明らかになりました。

### 4. 主要な発見

主要な発見を3点まとめます。

**発見1: 越谷市の特徴的パターン**

- イオンレイクタウン開業（2008年）後の14年間で
- 事業所数は15%減少（1,984→1,679事業所）
- 従業者数は7.5%増加（16,165→17,382人）
- **事業所の大型化と雇用維持の両立**を実現

**発見2: 産業構造の転換**

- 事業所数は2007→2012年で約30%減少
- その後2012→2021年は緩やかな変化
- 事業所の中規模化・効率化が進行

**発見3: データ収集方法論の確立**

- e-Stat APIとCSVの組み合わせ活用
- 超詳細地域コードレベル（API）と市区町村レベル（CSV）の使い分け
- 10都市×3時点の信頼性の高い時系列データを構築

## 質疑応答の想定質問・回答例

### Q1: なぜ10都市を選んだのか?

**A**: 3つの理由があります。第一に、全国5地域をバランスよくカバーするため。第二に、大型商業施設の開業時期が明確な都市を選定するため。第三に、e-Statでのデータ取得確実性が高い都市に絞るためです。Phase 1では実証可能性を重視し、10都市から開始しました。

### Q2: データ完全性が93%なのはなぜか?

**A**: 10都市×3年（2007/2012/2021）で期待レコードは30レコードですが、一部の都市で特定年次のデータが欠損しているため、現時点で28レコード、93%の完全性となっています。ただし、分析に必要な主要データは全て揃っており、統計分析には十分な品質です。なお、2014年APIデータも10レコード取得していますが、集計レベルが異なるため主分析からは除外しています。

### Q3: 越谷市の特徴的パターンの原因は?

**A**: 事業所数は15%減少しながらも従業者数が7.5%増加するという、他都市とは異なるパターンを示しています。仮説としては、イオンレイクタウンが日本最大級の施設であり、大規模雇用を創出する一方で、小規模店の一部が淘汰されたと考えられます。今後、開業年との相関分析やクラスター分析を通じて、詳細な要因を特定していきます。

### Q4: 2007-2012年の事業所数約30%減少の原因は?

**A**: 2つの要因が考えられます。第一に、2008年のリーマンショックによる経済的影響。第二に、大型商業施設の急速展開による小規模事業所の淘汰です。この期間は全国的に小売業の構造転換が起きた時期であり、今後、統計的検定により、どちらの影響が大きいか検証していきます。

### Q5: 2014年のデータを主分析から除外した理由は?

**A**: 2014年データはe-Stat API経由で取得したため、超詳細地域コードレベルの粒度となっており、市区町村レベルで集計されたCSVデータと直接比較できないためです。例えば、越谷市のCSV年度は約1,600～2,000事業所ですが、APIでは14,500事業所と10倍近い数値になります。これはデータの詳細度の違いであり、同一条件での時系列比較を優先するため、CSV年度のみを主分析対象としました。
