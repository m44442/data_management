# e-Stat 2017年データ取得手順書

## 🎯 取得対象

- **統計調査**: 2017年経済センサス-活動調査
- **対象都市**: 12都市（Phase 1 + 既存2都市）
- **対象産業**: 小売業全般

## 📊 検索手順

### Step 1: e-Stat基本検索

```
URL: https://www.e-stat.go.jp/
検索語: "2017年経済センサス活動調査 市区町村別 小売業"
```

### Step 2: 統計表特定

```
統計表名: 2017年経済センサス‐活動調査 産業別集計（市区町村別）
統計表ID: 0003348423
提供統計名: 経済センサス‐活動調査
提供分類1: 産業別集計
```

### Step 3: データ項目選択

```
表示項目:
- 事業所数
- 従業者数
- 売上（収入）金額
- 売場面積

産業分類:
- 各種商品小売業
- 織物・衣服・身の回り品小売業
- 飲食料品小売業
- 機械器具小売業
- その他の小売業
```

```

パラメータ:
- appId: [要申請]

```

## ⚠️ 注意事項

1. **ファイルサイズ**: 通常数MB程度
2. **ダウンロード時間**: 1-2分程度
3. **データ形式**: CSV、文字コードShift-JIS
4. **利用規約**: 政府統計データは学術利用OK
